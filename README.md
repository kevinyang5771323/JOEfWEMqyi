# 旅游景典可视化分析系统 python1080

## 项目概述

旅游景典可视化分析系统是一个基于Python和Django框架，前端使用VUE开发的Web应用。该系统旨在提供景点信息管理、旅游咨询发布、行程分享、个人收藏夹等一站式旅游信息服务。

## 技术栈

- 后端：Python、Django框架
- 前端：VUE
- 数据库：MySQL
- 开发工具：PyCharm

## 功能模块

### 前台管理

- 景点信息
- 旅游咨询
- 行程分享
- 个人中心
- 我的收藏

### 后台管理

- 景点信息管理
- 景点分类管理
- 酒店信息管理
- 行程分享管理
- 系统管理

## 系统角色

系统包含两类用户角色：

- 普通用户：浏览景点信息、查看旅游咨询、分享行程等。
- 管理员：全面管理后台数据，包括景点信息、分类、酒店信息和系统设置。

## 运行环境

- 操作系统：支持Linux、Windows等常见操作系统
- Python版本：3.x
- 数据库：MySQL 5.x 或更高版本

## 部署步骤

（以下步骤为一般性描述，具体部署请依据实际环境调整）

1. 安装Python环境
2. 安装MySQL数据库
3. 部署后端服务，配置数据库连接
4. 部署前端VUE项目
5. 配置Web服务器，如Apache或Nginx
